# 基于Java 11的常用代码库


`mvn install`

添加 Maven 库
----
    <dependency>
        <groupId>com.github.cckoolu</groupId>
        <artifactId>Lu-java</artifactId>
        <version>1.0-latest</version>
    </dependency>
----

### TimeStampUtil
